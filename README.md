# [Allday경주]

### 해당 서비스는 2024 경주 지역문제 해결 해커톤 [솔리온]의 프로젝트 입니다.

---
## 서비스 소개

### [allday경주]
- [allday경주]는 경주 거주민이나 대학생들에게 교육기회와 창업, 크라우드펀딩 서비스를 제공하는 서비스입니다.

---
## 기능
- 학습자 매칭
- 크라우드 펀딩
- 창업투자중개
